# Agent Rules: DO's and DONT's

## DO's
- **Keep it brief**: Talk/give short explanations when instructing or summarizing.
- **Selective summaries**: Summarize only where necessary.
- **Robust coding**: Keep changes safe and target-specific.

## DONT's
- **No browser testing**: Do not attempt to test on the browser.
- **No breaking or exploring**: Do not break code or do unnecessary exploration.
